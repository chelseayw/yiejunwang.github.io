Workspace

Digital, UI/UX
Proposed web application that would alievate the frustration of running multiple applications for various tasks, based off the aesthetic of Material Design. It allows users to create multiple workspaces for each of their tasks and organize applications they normally use between work and play. Each space allows users to select multiple apps to run at a time, streamlining the multitasking experience. I chose to base the design off the functionality of Google Docs and explore conventions that most people are comfortable using.
